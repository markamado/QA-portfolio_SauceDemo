# Страница оформления заказа (SauceDemo)

## Контактные данные (Contact)
- [ ] Поле "Email" принимает валидный email
- [ ] Поле "Email" не принимает невалидный формат (без @, домена и т.д.)
- [ ] Поле "Email" обязательно для заполнения
- [ ] Чекбокс "Email me with news and offers" можно включить/выключить

## Delivery (Доставка)
- [ ] Выбор страны (Country/Region) работает корректно
- [ ] Поле "First name" принимает ввод
- [ ] Поле "Last name" принимает ввод
- [ ] Поле "Address" принимает ввод
- [ ] Поле "City" принимает ввод
- [ ] Поле "Postal code" принимает валидный индекс
- [ ] Обязательные поля валидируются при пустом вводе
- [ ] Поля не принимают некорректные значения (если есть ограничения)

## Shipping method
- [ ] Блок "Shipping method" отображается
- [ ] Методы доставки появляются после заполнения адреса
- [ ] Можно выбрать способ доставки (если доступен)

## Payment (Оплата)
- [ ] Поле "Card number" принимает валидный номер карты
- [ ] Поле "Expiration date (MM/YY)" принимает корректный формат
- [ ] Поле "Security code" принимает только цифры
- [ ] Поле "Name on card" принимает ввод
- [ ] Чекбокс "Use shipping address as billing address" работает корректно
- [ ] Обязательные поля валидируются при пустом вводе

## Кнопка оплаты
- [ ] Кнопка "Pay now" неактивна при незаполненных обязательных полях
- [ ] Кнопка "Pay now" активна при заполненных обязательных полях
- [ ] При нажатии "Pay now" происходит попытка оформления заказа

## Сводка заказа
- [ ] Отображается добавленный товар
- [ ] Отображается название товара
- [ ] Отображается выбранный размер и цвет
- [ ] Отображается цена товара
- [ ] Отображается корректный subtotal
- [ ] Отображается стоимость доставки (если рассчитана)
- [ ] Отображается итоговая сумма (Total)
- [ ] Итоговая сумма рассчитывается корректно

## Промокод (Discount code)
- [ ] Поле "Discount code" принимает ввод
- [ ] Кнопка "Apply" активна при вводе значения
- [ ] Применение валидного промокода изменяет итоговую сумму (если применимо)
- [ ] При вводе невалидного промокода отображается ошибка

## Навигация 
- [ ] Кнопка возврата к корзине 

## UI проверки
- [ ] Верстка страницы не ломается
- [ ] Все элементы отображаются корректно
- [ ] Ошибки отображаются читаемо и понятно
- [ ] Поля и кнопки доступны для взаимодействия
