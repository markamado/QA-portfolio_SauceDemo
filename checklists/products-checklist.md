# Каталог товаров (SauceDemo)

## Страница каталога
- [ ] Отображается список товаров на странице
- [ ] У каждого товара отображается изображение
- [ ] У каждого товара отображается название
- [ ] У каждого товара отображается цена
- [ ] Товар кликабелен и открывает страницу товара

## Карточка товара
- [ ] Отображается название товара
- [ ] Отображается цена товара
- [ ] Отображается описание товара
- [ ] Доступен выбор размера
- [ ] Доступен выбор цвета
- [ ] Кнопка "Add to cart" отображается и доступна для нажатия

## Навигация
- [ ] Переход из каталога в карточку товара работает корректно
- [ ] Возврат на страницу каталога работает корректно

## UI проверки
- [ ] Изображения товаров отображаются корректно
- [ ] Текст не обрезается и читаем
- [ ] Верстка страницы не ломается
- [ ] Отображается статус "Sold out" для недоступных товаров
