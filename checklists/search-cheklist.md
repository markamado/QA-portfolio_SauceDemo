# Поиск (SauceDemo)

## Функциональные проверки
- [ ] Поиск по существующему товару (например, "Black heels") возвращает корректный результат
- [ ] Поиск по частичному совпадению (например, "Black") возвращает релевантные товары
- [ ] Поиск несуществующего товара отображает сообщение об отсутствии результатов
- [ ] Поиск чувствителен к регистру (если да — работает корректно / если нет — игнорирует регистр)
- [ ] Поиск по пустому запросу не вызывает ошибку
- [ ] Поиск по спецсимволам (например, "@#$%") обрабатывается корректно
- [ ] Поиск по числовым значениям (если применимо) работает корректно

## UI проверки
- [ ] Поле поиска отображается корректно
- [ ] Placeholder (например, "Search") отображается
- [ ] Введенный текст отображается корректно
- [ ] Кнопка поиска (если есть) отображается и активна
- [ ] Результаты поиска отображаются в корректном формате (карточки товаров)
- [ ] Верстка не ломается при длинных поисковых запросах

## Навигация
- [ ] При клике на найденный товар происходит переход в карточку товара
- [ ] Очистка поля поиска возвращает пользователя к полному списку товаров (если реализовано)

## Edge cases
- [ ] Ввод очень длинного текста (например, 100+ символов) не ломает поиск
- [ ] Быстрое удаление и ввод текста не вызывает багов
- [ ] Поиск работает после обновления страницы

## Производительность
- [ ] Результаты поиска отображаются без значительных задержек
