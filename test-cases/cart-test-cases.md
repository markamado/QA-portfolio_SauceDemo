## Тест-кейсы корзины (SauceDemo)

### TК-001: Добавление товара в корзину 

**Предусловие:** пользователь находится на странице каталога https://sauce-demo.myshopify.com/collections/all

**Шаги:**
1. Открыть страницу каталога  
2. Нажать на товар "Black heels"  
3. Нажать "Add to cart"  
4. Перейти в корзину  

**Ожидаемый результат:**  
Товар "Black heels" отображается в корзине с корректным названием, ценой и количеством = 1  

### TК-002: Удаление товара из корзины

**Предусловие:** в корзину добавлен товар  

**Шаги:**
1. Открыть страницу корзины  
2. Нажать на кнопку "X"  

**Ожидаемый результат:**  
Товар удален из корзины  

### TК-003: Изменение количества товара в корзине 

**Предусловие:** в корзину добавлен товар  

**Шаги:**
1. Открыть страницу корзины  
2. Ввести в поле "Qty" значение 2  
3. Нажать Enter  

**Ожидаемый результат:**  
Количество товара изменяется на 2  

### TК-004: Перерасчет цены после изменения количества товара

**Предусловие:** в корзину добавлен товар  

**Шаги:**
1. Открыть страницу корзины  
2. Ввести в поле "Qty" значение 2  
3. Нажать Enter  

**Ожидаемый результат:**  
Общая стоимость товара пересчитывается пропорционально количеству (увеличивается в 2 раза)  

### TК-005: Переход на страницу оформления заказа

**Предусловие:** в корзину добавлен товар  

**Шаги:**
1. Открыть страницу корзины  
2. Нажать на кнопку "Check out"  

**Ожидаемый результат:**  
Происходит переход на страницу оформления заказа  

### TК-006: Переход из корзины на страницу каталога 

**Предусловие:** открыта страница корзины  

**Шаги:**
1. Открыть страницу корзины  
2. Нажать на кнопку "Continue Shopping"  

**Ожидаемый результат:**  
Происходит переход на страницу каталога  

### TК-007: Удаление последнего товара из корзины

**Предусловие:** в корзину добавлен один товар  

**Шаги:**
1. Открыть страницу корзины  
2. Нажать на кнопку "X"  

**Ожидаемый результат:**  
В корзине отсутствуют товары, отображается сообщение "It appears that your cart is currently empty! Continue Shopping."  

### TК-008: Ввод некорректного количества товара

**Предусловие:** в корзину добавлен один товар  

**Шаги:**
1. Открыть страницу корзины  
2. В поле "Qty" ввести значение -1  

**Ожидаемый результат:**  
Поле "Qty" не принимает значения меньше 1 или автоматически возвращает корректное значение  

### TК-009: Корректный расчет общей суммы корзины

**Предусловие:** в корзину добавлено несколько товаров  

**Шаги:**
1. Открыть страницу корзины  

**Ожидаемый результат:**  
Общая сумма корзины равна сумме цен всех товаров с учетом их количества

### TК-010: Сохранение товаров в корзине после обновления страницы

**Предусловие:** в корзину добавлен товар  

**Шаги:**
1. Открыть страницу корзины  
2. Обновить страницу браузера  

**Ожидаемый результат:**  
Товары остаются в корзине, данные не теряются  

### TК-011: Сохранение товаров в корзине после повторного открытия сайта

**Предусловие:** в корзину добавлен товар  

**Шаги:**
1. Добавить товар в корзину  
2. Закрыть вкладку браузера  
3. Открыть сайт заново  
4. Перейти в корзину  

**Ожидаемый результат:**  
Ранее добавленные товары отображаются в корзине
