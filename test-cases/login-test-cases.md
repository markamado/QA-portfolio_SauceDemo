## Тест-кейсы авторизации (SauceDemo)

### - TК-001: Успешная авторизация с валидными данными

**Предусловие:** Пользователь зарегистрирован (email: markbalde@mail.ru, пароль: Balde10)

**Шаги:**
1. Открыть страницу логина https://sauce-demo.myshopify.com/account/login 
2. Ввести в поле email "markbalde@mail.ru"
3. Ввести в поле пароля "Balde10"
4. Нажать кнопку "Sign in"

**Ожидаемый результат:**
Пользователь успешно авторизован и перенаправлен в личный кабинет
