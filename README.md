# QA Portfolio — SauceDemo

## 📌 О проекте
Тестирование демо интернет-магазина на платформе Shopify:  
https://sauce-demo.myshopify.com/

Цель проекта — продемонстрировать навыки ручного тестирования:
- составление чек-листов
- написание тест-кейсов
- поиск и оформление багов

---

## 🔍 Области тестирования

В рамках проекта были протестированы следующие модули:

- Каталог товаров
- Карточка товара
- Корзина
- Поиск
- Оформление заказа (Checkout)

---

## 🧪 Что сделано

### ✅ Чек-листы
Чек-листы покрывают:
- функциональность
- UI проверки
- навигацию
- граничные значения
  
---
### ✅ Тест-кейсы
Тест-кейсы включают:
- предусловия
- пошаговые действия
- ожидаемый результат

Покрыты основные пользовательские сценарии:
- добавление товара в корзину
- изменение количества товара
- удаление товара
- переходы между страницами
- поиск товаров
---

### 🐞 Баг-репорты
В процессе тестирования были найдены и оформлены баги, включая:

- проблема с переходом в корзину (loader без перехода)
- некорректное поведение поля комментария в корзине
- UI-проблемы (кнопка "Update" выглядит неактивной)
- некорректная обработка пустого поиска
---

## 🛠 Используемые подходы

- Black-box тестирование
- UI/UX тестирование
- Тестирование граничных значений
- Проверка пользовательских сценариев
---

## 📈 Результат

Данный проект демонстрирует:

- умение структурировать тестовую документацию  
- понимание пользовательских сценариев  
- навыки поиска и оформления багов  
- внимание к деталям  
---

## 🚀 Как использовать

1. Открыть тестируемый сайт  
2. Перейти в нужный раздел проекта (checklists / test-cases / bug-reports)  
3. Использовать документацию для воспроизведения сценариев
